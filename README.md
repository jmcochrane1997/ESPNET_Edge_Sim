### This is the code base for CMU 11692 project: SIMULATING A SPEECH-TO-SPEECH SYSTEM IN EDGE COMPUTING 

## This GitHub consists of the ESPNET GitHub with an additional directory in espnet2 called edgeSim. This new directory contains the source code that simulates an OWSM 3.1 --> QA LLM --> Tiny TTS Model on the NETCAST ONN from https://www.science.org/doi/10.1126/science.abq8271

## The intended runtime environment for this source code is google colab. Please see our notebook at: https://colab.research.google.com/drive/1nYT4GLn9qClyj-My-_rnlxVzVkff8qNm?usp=sharing
